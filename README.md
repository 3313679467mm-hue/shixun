# 王者荣耀智能客服机器人

这是根据 `yaoqiu.docx` 实现的实训基础达标版项目，提供后台知识库管理平台和前端对话机器人。

## 已实现功能

- 知识库创建、列表展示、分类、负责人、状态与文档数量统计
- 知识库按名称/描述/负责人/部门筛选，按名称/部门/创建时间排序
- 知识库编辑，支持修改名称、描述、分类、所属部门、负责人和 Embedding 模型
- 知识库逻辑删除与物理删除，物理删除会同步删除文档和向量数据
- 知识库克隆，可复制已有知识库配置、文档和向量元数据
- 文档上传，支持 `txt`、`md`、`docx`、`pdf`
- 文档批量上传、上传进度、解析、文本清洗、自动分块、本地索引生成
- 文本分块参数配置，支持原文预览、文本块预览和关键词筛选
- 文档列表按名称筛选、按名称/上传时间排序
- 文档在线编辑，保存后自动重新分块并更新向量索引
- 文档单篇删除和批量删除，同步删除对应向量数据
- 向量管理模块：支持手动重新生成向量，展示批量生成进度和每篇文档生成结果
- 向量预览：可查看文本块对应的向量维度、相似度得分和关键词向量预览
- 向量优化：支持向量归一化和按文本相似度去重，删除重复文本块向量并保留原始文档
- 系统管理模块：支持 Embedding 模型列表、模型路径、维度、描述和默认模型设置
- 系统参数配置：支持默认文本拆分参数、上传大小限制、向量库存储路径、接口超时和缓存策略配置
- 权限管理模块：支持首次初始化管理员、账号密码登录、记住登录状态、退出登录和 token 校验
- 用户管理：支持创建、编辑、删除用户，设置用户名、密码、邮箱、部门、状态、角色和可访问知识库范围
- 角色管理：内置超级管理员、知识库管理员、普通用户，支持自定义角色和权限矩阵
- 接口层 RBAC 校验：知识库、文档、向量、系统配置、日志和权限管理接口按角色权限拦截
- 基于本地关键词向量的语义检索、Top K 和阈值配置
- 单轮问答、多轮上下文、来源文档标注、相似度展示
- 检索无结果兜底提示、转人工客服触发
- 对话日志记录、查询展示、响应耗时统计
- 本地 SQLite 持久化存储，重启后数据保留

## 启动方式

Windows 推荐：

```powershell
.\start.ps1
```

通用方式：

```powershell
python app.py
```

然后打开：

```text
http://127.0.0.1:8000
```

首次打开会要求初始化超级管理员账号。请自行设置用户名和密码，系统不会生成固定默认密码。

## 项目结构

```text
app.py              后端服务与本地检索逻辑
static/index.html   页面结构
static/styles.css   页面样式
static/app.js       前端交互逻辑
data/               运行后自动生成，保存数据库和上传文档
data/config.json    系统参数配置
models/             可选，本地 Embedding 模型目录
```

## 演示建议

首次启动会自动创建一个“王者荣耀客服知识库”和一份演示问答文档。可以直接提问：

- 点券不到账怎么办
- 怎么改实名认证
- 活动奖励没到账
- 我要转人工客服

也可以在“知识库管理”中创建新知识库并上传自己的文档，再切回“对话机器人”测试问答效果。
