
## 一、韩信（刺客·打野）
### 1. 英雄定位与核心特点
韩信定位为刺客型打野，核心特点是高机动性、强拉扯能力与快速刷野效率，拥有多段位移与控制技能，擅长反野、带线牵制与收割残血。该英雄操作上限高，依赖经济优势打出压制效果，缺点是身板较脆，容错率低，进场时机失误容易被集火秒杀。

### 2. 技能深度解析
#### 被动技能——杀意之枪
韩信第四次普通攻击会将敌人击飞0.5秒，同时技能命中目标后会提升自身50%攻击速度，持续3秒。
使用技巧：刷野与对拼时尽量叠满被动打出击飞控制，利用攻速加成提升输出效率；gank时可先借助野怪叠好被动，再进场打出控制链。

#### 一技能——无情冲锋
韩信向指定方向冲锋，对路径上的敌人造成物理伤害，5秒内可释放第二段，第二段命中敌人会造成击飞效果。
使用技巧：第一段用于进场或拉近身位，第二段留作控制、追击或撤离；穿墙时需把控距离，可利用该技能翻越峡谷大部分墙体；团战中不要一次性交完两段位移，保留逃生或追击空间。

#### 二技能——背水一战
韩信向后跳跃，下一次普通攻击变为横扫，对敌人造成物理伤害；若攻击命中敌人背部，会触发暴击效果。
使用技巧：该技能是韩信核心输出技能，背击暴击可爆发高额伤害；连招中通常先用二技能调整身位，保证普攻命中敌人背部；也可利用后跳位移拉开距离，实现拉扯反打。

#### 三技能——国士无双
韩信朝指定方向挥舞长枪，持续对范围内敌人造成多段物理伤害，最后一段会击飞范围内所有敌人；技能释放期间韩信处于霸体状态，获得30%免伤。
使用技巧：大招可用于规避控制、抵挡爆发伤害，同时打出范围击飞；团战中尽量让大招覆盖更多敌方英雄，配合被动与一技能形成连续控制；避免在敌方关键控制技能交出前盲目进场开大。

### 3. 出装与铭文推荐
#### 常规出装方案
追击刀锋（升级为浴火之怒）→ 抵抗之靴 → 暗影战斧 → 无尽战刃 → 宗师之力 → 名刀·司命
后期可将名刀替换为贤者的庇护，实现秒换三装；若敌方坦度较高，可将宗师之力换为碎星锤。

#### 逆风出装方案
追击刀锋 → 抵抗之靴 → 暗影战斧 → 冰痕之握 → 暴烈之甲 → 魔女斗篷
逆风局降低纯输出占比，补充防御属性，提升带线与拉扯时的生存能力，依靠装备被动打持续伤害。

#### 铭文搭配
7异变+3红月、6隐匿+4夺萃、10鹰眼
属性效果：提供物理攻击、物理穿透、攻速与吸血能力，16.6%攻速档位适配韩信普攻节奏，少量吸血可提升刷野续航，减少回城次数。

### 4. 核心连招技巧
- **基础连招：21A31**
  二技能起手调整背击位置，一技能第一段进场接普攻触发暴击，开启大招打出伤害与击飞，最后用一技能第二段追击或撤离。适用于常规gank与单挑场景。
- **蹲草秒人连招：2A13A1**
  蹲伏草丛时，先二技能存好横扫普攻，敌人靠近后先打出背击暴击，再接一技能击飞，开大补伤害，最后用第二段一技能收尾。适用于蹲守脆皮英雄。
- **拉扯消耗连招：1A2A1**
  一技能第一段近身普攻，二技能拉开距离同时打出横扫普攻，再用一技能第二段撤离。用于对线消耗、试探敌方技能，避免硬拼。
- **无影步技巧**
  利用一技能或二技能位移进入草丛瞬间，衔接普攻，可消除自身视野，实现隐身进场或撤离，常用于反野与蹲人。

### 5. 打野与运营思路
#### 刷野节奏
常规开局优先蓝buff开，快速刷完蓝区三组野怪到达四级，根据线上兵线与敌方状态选择gank路线；若敌方打野前期弱势，可直接反红buff打乱对方节奏。韩信刷野速度快，需利用时间差多刷野、多反野，拉开经济差距。

#### gank思路
优先抓控制多、兵线靠前的路线，gank前先观察敌方闪现与关键技能是否交出；抓对抗路可配合队友越塔，抓发育路优先蹲草丛打背击。gank失败不要长时间逗留，立刻返回野区刷野，避免经济落后。

#### 带线牵制
中后期以带线牵制为主，利用高机动性牵制敌方多人，给队友争取推塔、拿龙的时间；兵线带过河道后及时撤离，避免被多人包围。

### 6. 团战打法思路
韩信团战核心任务是切后排与收割残血，避免正面硬接敌方技能。
- 进场时机：等敌方前排位移与控制技能交出后，从侧面或绕后进场，优先攻击敌方射手、法师等核心输出位。
- 输出方式：进场后快速打出一套爆发，若无法秒杀立刻用位移撤离，等待下一轮技能CD，切忌原地站撸。
- 收割阶段：敌方出现残血后，利用多段位移追击收割；团战溃败时，可利用位移断兵线、偷塔牵制，延缓敌方推进节奏。
- 注意事项：团战中保留至少一段位移，用于躲避关键控制或撤离；名刀、复活甲等保命装备要及时更换，提升容错率。

## 二、铠（战士·对抗路）
### 1. 英雄定位与核心特点
铠定位为战士型对抗路英雄，兼具爆发伤害与续航抗压能力，开启大招后攻防属性大幅提升，有“铠皇”之称。该英雄操作门槛低，单挑能力极强，适合新手上手；缺点是缺乏位移技能，腿短容易被风筝，依赖大招持续时间打输出。

### 2. 技能深度解析
#### 被动技能——修罗之魂
铠的普攻与二技能若只命中一个单位，会对该目标造成额外50%伤害。
使用技巧：对线与单挑时尽量保证技能只命中单个敌人，最大化伤害；团战中优先锁定单个目标输出，避免伤害分散。

#### 一技能——回旋之刃
铠向指定方向投掷刀刃，对路径上的敌人造成物理伤害与减速效果，刀刃回弹后为铠回复生命值与冷却缩减，同时提升自身移速。
使用技巧：对线期核心消耗与续航技能，可借助兵线回弹刀刃，保证自身血量健康；追击与拉扯时用一技能减速敌人，提升留人能力。

#### 二技能——极刃风暴
铠向前挥砍两次，第二次挥砍会将敌人击飞，下一次普通攻击变为冲砍，附带额外伤害与位移效果。
使用技巧：二技能是核心控制与输出技能，配合被动可打出高额单体伤害；冲砍普攻可用于拉近身位，弥补铠位移不足的缺陷；对线换血时，可先二技能击飞接普攻，再用一技能减速撤离。

#### 三技能——不灭魔躯
铠召唤魔铠，提升自身攻击力、移速、格挡效果，持续8秒，开启瞬间对周围敌人造成法术伤害。
使用技巧：大招是铠的核心强势期，开启后攻防属性大幅提升，可正面硬接敌方伤害；尽量在靠近敌人前开启大招，避免大招持续时间浪费在赶路上；团战与单挑前提前开大招，保证输出拉满。

### 3. 出装与铭文推荐
#### 常规输出出装
抵抗之靴 → 暗影战斧 → 宗师之力 → 无尽战刃 → 暴烈之甲 → 名刀·司命
该出装爆发伤害高，适配单带与切后排打法，适合优势局与敌方脆皮较多的阵容。

#### 半肉抗压出装
抵抗之靴 → 暗影战斧 → 冰痕之握 → 不祥征兆 → 魔女斗篷 → 碎星锤
逆风局或敌方物理输出较多时使用，兼顾伤害与坦度，可在前排抗伤，同时依靠装备被动打持续输出。

#### 铭文搭配
10异变、10隐匿、10鹰眼
全穿透铭文最大化前期伤害，适配铠的技能爆发特性，对线期可快速压低敌方血线；若追求续航，可将部分隐匿替换为夺萃。

### 4. 核心连招技巧
- **基础单挑连招：3A2A1A**
  提前开启大招强化属性，普攻起手，接二技能击飞+冲砍普攻，再用一技能减速补伤害，最后衔接普攻收尾。适用于四级后对线单杀。
- **蹲草秒人连招：32A1A**
  蹲伏草丛，敌人靠近后先开大招，立刻二技能击飞打出冲砍普攻，接一技能减速，持续普攻追击。适合蹲守脆皮法师、射手。
- **团战进场连招：132A**
  一技能减速敌人后开启大招，二技能突进击飞目标，衔接普攻打爆发。适用于侧面进场切后排。

### 5. 对线思路
#### 前期对线（1-4级）
一级优先学习一技能，借助兵线回弹回血，保证线上续航；二级学习二技能后，可利用草丛卡视野，二技能击飞接普攻打一套消耗，打完立刻后撤。四级前以发育为主，避免强行换血，等大招解锁后再找单杀机会。

#### 四级强势期
四级解锁大招后是铠的第一个强势点，可主动找敌方对拼。换血时先开大招，利用二技能击飞+被动单体增伤，快速压低敌方血线，敌方血量低于半血即可尝试越塔单杀。

#### 兵线处理
优先清理兵线再支援，避免因支援漏兵导致经济落后；清线后可入侵敌方野区偷小野，或支援中路、野区打小规模团战。

### 6. 团战打法思路
铠团战有两种打法，根据阵容与局势灵活选择：
- **绕后切C打法**：适合优势局，从侧面或绕后位置进场，目标锁定敌方射手、法师，开启大招后用二技能突进，一套爆发秒杀核心输出。打完一套若无法继续输出，立刻后撤等待技能CD。
- **前排抗伤打法**：适合逆风或缺前排的阵容，开启大招后顶在队伍前方，吸收伤害的同时用一技能减速、二技能控制，保护己方后排；利用大招格挡效果扛住敌方爆发，为队友创造输出环境。
- 注意事项：避免被敌方长手英雄持续风筝，尽量在地形狭窄的区域开战，减少拉扯空间；关键控制技能留给敌方核心输出，不要轻易交掉二技能击飞。

## 三、安琪拉（法师·中路）
### 1. 英雄定位与核心特点
安琪拉定位为爆发型法师，核心特点是超高瞬间爆发与范围控制，大招伤害极高，可快速秒杀脆皮与半肉英雄。该英雄操作简单，上手门槛低，是新手法师首选；缺点是没有位移技能，自保能力弱，依赖蹲草与预判打伤害，技能空窗期输出能力大幅下降。

### 2. 技能深度解析
#### 被动技能——咒术火焰
安琪拉的技能会对敌人叠加灼烧印记，最多叠加3层，叠满后会造成额外法术伤害并降低敌方法术防御。
使用技巧：尽量让多个技能命中同一目标，叠满被动打出爆发伤害；大招持续照射可快速叠加被动，提升整体输出。

#### 一技能——火球术
安琪拉向指定方向发射五颗火球，每颗火球对敌人造成法术伤害，多个火球命中同一目标时，后续火球只造成30%伤害。
使用技巧：对线清线核心技能，近距离贴脸释放可打出高额伤害；远距离消耗时瞄准敌方英雄走位预判释放；清线时对准兵线中心，保证快速清理兵线。

#### 二技能——混沌火种
安琪拉向指定方向释放火种，命中敌人后造成法术伤害与眩晕效果，同时在地面留下火焰区域，持续造成伤害与减速。
使用技巧：核心控制技能，也是大招命中的前提，需要预判敌方走位；蹲草时先手二技能眩晕，接大招可直接秒杀；可将二技能放在敌方逃跑路径上，实现封路留人。

#### 三技能——炽热光辉
安琪拉朝指定方向持续发射光束，造成多段法术伤害，释放期间获得护盾，可免疫控制效果；可通过转动方向调整光束位置。
使用技巧：安琪拉核心输出技能，伤害极高，配合二技能眩晕可打满全部伤害；释放期间有护盾与免控，可抵挡敌方技能与控制；大招可转动方向追击敌人，但转动速度较慢，需提前预判走位。

### 3. 出装与铭文推荐
#### 常规爆发出装
疾步之靴 → 回响之杖 → 痛苦面具 → 博学者之怒 → 虚无法杖 → 辉月
该出装爆发伤害拉满，支援速度快，适配安琪拉蹲草秒人打法；辉月可替换为贤者之书进一步提升伤害，或换成梦魇之牙针对敌方回血英雄。

#### 半肉自保出装
抵抗之靴 → 回响之杖 → 痛苦面具 → 博学者之怒 → 极寒风暴 → 魔女斗篷
敌方刺客较多、自保压力大时使用，补充防御属性与冷却缩减，提升生存能力，同时保留核心输出强度。

#### 铭文搭配
10梦魇、10轮回、10心眼
提供法术攻击、法术穿透与法术吸血，兼顾输出与续航，适配中路对线消耗场景。

### 4. 核心连招技巧
- **基础秒人连招：231**
  二技能先手眩晕目标，立刻开启大招打满伤害，最后补一技能收尾。适用于蹲草、近距离遭遇战，是安琪拉最常用的秒杀连招。
- **远距离消耗连招：123**
  一技能先消耗压低血线，预判走位接二技能眩晕，最后开大收割。适用于对线消耗、远距离支援。
- **反手自保连招：321**
  敌方刺客突进时，先开大招获得护盾与免控扛伤害，再用二技能眩晕反打，最后一技能补伤害。适用于被突袭时的反打场景。

### 5. 对线思路
#### 前期对线
一级学习一技能，快速清理第一波兵线抢升二级；二级学习二技能后，可尝试预判敌方走位眩晕打消耗。对线期以清线为主，清完兵线后可支援上下路，或入侵敌方野区干扰打野节奏。

#### 支援节奏
安琪拉清线速度快，清完中路兵线后要及时游走支援，利用蹲草优势配合边路队友击杀敌人；支援时走河道草丛，避免暴露视野，提升gank成功率。

#### 视野利用
安琪拉核心打法是蹲草伏击，可蹲守中路两侧草丛、蓝buff草丛、龙坑草丛等位置，等敌方脆皮经过时一套秒杀。

### 6. 团战打法思路
- **站位选择**：团战中站在己方后排位置，不要贸然靠前，利用技能射程输出；也可提前蹲守团战区域的侧边草丛，等敌方后排进场后突袭。
- **输出优先级**：优先攻击敌方射手、法师等核心输出，二技能尽量留给敌方突进刺客，保护自身与己方后排。
- **大招释放时机**：等敌方交出位移技能后再开大，避免大招空转；团战混乱时，大招可对准敌方人群，利用范围伤害压低多人血线。
- 自保技巧：辉月装备可在大招结束后使用，躲避敌方爆发伤害；被突进时不要慌张，先开大招扛伤，再用二技能反打，利用护盾与免控提升反杀概率。

## 四、后羿（射手·发育路）
### 1. 英雄定位与核心特点
后羿定位为站桩输出型射手，核心特点是持续输出能力强、攻速快、普攻附带分裂箭效果，后期装备成型后团战输出极高。该英雄操作简单，适合新手练习走A；缺点是没有位移，自保能力极差，极度依赖队友保护，容易被刺客针对。

### 2. 技能深度解析
#### 被动技能——惩戒射击
后羿普攻命中敌人会提升自身攻速，最多叠加3层；叠满后普攻会分裂成三支箭矢，造成额外物理伤害。
使用技巧：对线与团战前先借助小兵或野怪叠满被动，保证输出最大化；走A过程中保持普攻节奏，避免被动层数中断。

#### 一技能——多重箭矢
后羿强化自身普攻，短时间内普攻可同时攻击多个目标，对主目标造成额外伤害。
使用技巧：核心输出技能，团战时开启可快速清线与打范围伤害；单挑时开启提升单体输出效率；推塔、打龙时开启可大幅提升速度。

#### 二技能——落日余晖
后羿在指定位置召唤火焰区域，对范围内敌人造成物理伤害与减速效果，中心区域会造成额外伤害与减速。
使用技巧：可用于清线、消耗与留人，对线期可放在敌方补刀位置，逼迫敌方走位；追击时放在敌人逃跑路径上，提升留人能力；被突进时放在脚下，减速敌方刺客创造逃生空间。

#### 三技能——灼日之矢
后羿向指定方向发射火焰大鸟，命中第一个敌方英雄后造成物理伤害与眩晕效果，飞行距离越远，眩晕时间越长，同时会暴露目标周围视野。
使用技巧：全图支援技能，可用于远程开团、留人、探视野；远距离命中敌方核心可配合队友完成击杀；团战前可先用大鸟开团，打敌方一个措手不及；也可用于打断敌方回城、探测草丛视野。

### 3. 出装与铭文推荐
#### 常规攻速流出装
急速战靴 → 闪电匕首 → 无尽战刃 → 泣血之刃 → 破晓 → 贤者的庇护
标准站桩射手出装，兼顾攻速、暴击、吸血与穿透，后期输出与续航兼备，是最通用的出装方案。

#### 攻速流备选出装
急速战靴 → 影刃 → 无尽战刃 → 破晓 → 泣血之刃 → 不祥征兆
敌方刺客较多时，将闪电匕首换为影刃，补充暴击移速，最后一件补防御装提升生存能力。

#### 铭文搭配
7红月+3无双、5夺萃+5狩猎、10鹰眼
提供攻速、暴击与吸血属性，卡好前期攻速档位，提升对线手感；少量吸血可保证前期续航，减少回城次数。

### 4. 核心操作技巧
- **走A输出**：利用普攻间隔小幅移动走位，躲避技能的同时保持输出节奏，避免原地站桩被技能命中。
- **远程开团**：根据小地图预判敌方走位，远距离释放大招眩晕核心目标，配合队友先手开团。
- **反手拉扯**：被敌方刺客突进时，先开二技能放在脚下减速，边走边普攻输出，配合辅助控制反打；不要一味逃跑，边打边退收益更高。

### 5. 对线思路
#### 前期发育
后羿前期伤害较低，以发育为主，避免强行换血。一级学习二技能，快速清理兵线；辅助配合时可尝试压塔，没有辅助保护时猥琐塔下，防止被敌方打野gank。

#### 对线消耗
借助兵线叠满被动后，用一技能强化普攻消耗敌方射手；二技能瞄准敌方补刀位置释放，逼迫敌方走位，同时压低血线。敌方刺客消失时立刻后撤，不要贪线。

#### 资源争夺
河道之灵、暴君、主宰等资源团，优先保证自身安全，在后排输出；不要为了抢资源贸然靠前，避免被开团秒杀。

### 6. 团战打法思路
- **站位原则**：始终站在己方辅助与前排身后，保持安全输出距离，不要脱节；优先选择有墙体、草丛的位置输出，方便遮挡技能。
- **输出顺序**：优先攻击能打到的目标，不要强行切后排导致自身暴毙；敌方刺客突进时，优先集火反杀刺客，再继续输出前排。
- **大招使用**：团战开启前可远距离放大招先手开团；团战过程中若敌方刺客未出现，保留大招用于反手控制突进的敌人。
- 注意事项：时刻关注敌方刺客位置，闪现与保命装备不要过早交出；团战中保持走A节奏，利用吸血装备维持血量，不要轻易回城。
